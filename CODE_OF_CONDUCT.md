# Code of Conduct

## Our Pledge

We are committed to a respectful, professional, and harassment-free project environment.

## Expected Behavior

- Be direct, constructive, and respectful.
- Focus feedback on code, evidence, documentation, and user impact.
- Assume good intent while still asking for clear technical reasoning.
- Respect maintainers' time and project scope.

## Unacceptable Behavior

- Harassment, insults, threats, or discriminatory language.
- Publishing private information without permission.
- Repeated off-topic, spammy, or disruptive behavior.
- Pressuring maintainers to accept changes that weaken security, evidence quality, or user trust.

## Enforcement

Maintainers may remove comments, close issues, block users, or reject contributions that violate this code of conduct.
